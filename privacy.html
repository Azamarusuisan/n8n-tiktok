プライバシーポリシー

本アプリ（以下「本アプリ」）は TikTok Content Posting API を使用し、 動画を下書きに投稿するサービスを提供します。
以下、ユーザー情報の取扱いについて説明します。

1. 取得・利用する情報

  • TikTok アカウント識別子（OpenID） – 認可管理に使用
  • アクセストークン – 動画下書き投稿の API 呼び出し時のみ使用
  • アップロード対象動画ファイル – TikTok へのアップロードに使用

2. 利用目的

取得した情報は、ユーザーが指定した動画を下書きとして投稿する目的のみに利用します。

3. 第三者提供

取得した情報を第三者へ提供することはありません。

4. 保存期間

アクセストークンは TikTok が定める有効期限終了後、自動的に無効化されます。動画ファイルはアップロード完了後速やかに削除します。

5. ユーザーによる許可の取消し

ユーザーは TikTok の「連携アプリ管理」画面から本アプリの認可をいつでも取り消すことができます。

6. お問い合わせ

データ取扱いに関するお問い合わせは example@example.com までご連絡ください。
